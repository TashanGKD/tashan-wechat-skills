# 活动预告｜Agent for Science 结营路演：从学习到共创，构建智能科研新生态

![](https://mmbiz.qpic.cn/mmbiz_jpg/YfcILJIWTS9A4pEBUFicHEPhpoxm2wINsa6tuAAibLy4UFWZqa3R7ibBBSFgpV5nVbT37FvMWWgibsLuym4zhl7MvA/640?wx_fmt=jpeg)

历时一月，从零到一；

结营在即，从共创到展示。

中国科学院大学首期「Agent for Science」实训营即将迎来高光收官。这不仅是一次学习成果的集中展示，更是一次面向未来的智能科研生态的共建探索。自实训营启动以来，我们共同完成了从理论到实践、从工具到系统的完整能力构建，现在，我们即将见证这些能力的集体呈现与价值延伸。

**📍 我们一同走过的「四步走」旅程**

\

\

\

**Step 1：从模型到智能体，建立“认知”与“执行”基础**

- 我们认识科研模型的工作原理，掌握领域知识对齐的基本方法；
- 我们理解科研智能体如何执行科研任务，完成第一个可调用工具的科研 Agent；
- 在吴泰霖研究员的前沿分享中，我们看到了AI驱动科研的广阔图景。

  \

**Step 2：打造专属“知识库”和“AI科研工作站”**

- 我们建立了个人科研知识库，让AI的回答有据可依、有“记忆”可循；
- 我们亲手搭建了一体化科研IDE平台，将文献管理、代码编写、工具调用整合于一处；
- 胡楠教授的讲座，让我们重新思考了智能体在社会科学中的角色与边界。

  \

**Step 3：完成自动化科研工作流闭环，从专属工具到闭环探索**

- 我们学会了将研究方法封装为可复用工具，让个人代码成为团队资产；
- 我们设计了第一个人机协作科研流程，让AI承担从问题分析到初步验证的全过程；
- 林佳本研究员展示了AI如何重塑天文观测，为我们提供了最直观的“AI Scientist”范例。

  \

**Step 4：深化专项研讨与项目孵化，构建可持续协作模式**

- **开展3次经管领域深度研讨，探索长期孵化路径**

  围绕反垄断、数字经济等真实场景，组织跨学科研讨，形成“问题-方法-系统”联动机制，为经管领域智能体应用奠定可持续的研究基础。
- **提供个性化项目辅导，助力路演成果优化与展示**

  针对9支路演团队开展4次个性化辅导，优化项目呈现逻辑与技术实现路径，确保成果可演示、可复现、可深化，为结营路演做好全面准备。
- **建立跨领域协作机制，形成可持续孵化模式**\

  通过专项研讨与项目孵化，初步构建起跨学科、跨院所的协作网络，为后续可持续的智能科研生态建设奠定组织基础。

**🚀 即将到来的「结营路演」：展示与起点**

\

\

\

本次结营路演定位为“学习成果展示 + 科研能力成长”的综合环节。 我们将集中呈现四周课程成果，识别可孵化对象，并为构建未来跨学科研讨与智能科研实践的持续生态迈出第一步。

\

**🔍 活动核心环节：**

**\**

**（一）精选路演（Featured Presentations）**

9组经过筛选的学员代表将进行现场成果展示，每组10分钟（6分钟汇报+4分钟问答）。项目涵盖：

向上滑动阅览

**1️⃣国际贸易合规多智能体系统。**

**目标：**动态整合各国贸易法规与出口管制政策，实现政策差异分析、违规模式提炼及企业合规预警。

**技术手段：**依托爬虫技术采集法规、处罚案例等数据，经集成对比后生成合规报告与风险预测。

**预期结果：**输出精准合规判断与任务清单，为企业出口贸易提供决策支撑。

\

**2️⃣反垄断案件要素提取与类案检索智能体**

**核心功能与场景：**基于 GraphRAG 技术，打造案件要素提取与类案检索智能体，自动拆解案件核心要素、构建法律知识图谱。

**预期实现：**检索准确率提升10%以上，大幅缩短检索时间，减少人工筛选成本，为法律从业者提供精准、高效的类案参考，助力司法办案质效提升。

\

3️⃣**太阳物理AI Scientist**

**核心功能：**利用 MCP 协议连接 NASA 实时数据接口与外部专家智能体。通过“科学家-数据员-程序员”多角色协作，实现从文献调研、CME/GST 数据对齐分析到自动预测报告生成的流程。主要应用于太阳活动对地影响的初探性科研辅助及自动化数据清洗场景。

**预期目标：**在实训营期间，旨在跑通最小可行性路径，验证 AI 自主调用工具处理复杂物理数据的可行性。

\

**4️⃣多智能体狼人杀行为仿真**

**目标：**构建可复现、可对比的多智能体推理实验环境，用于分析大模型在信息不完全情境下的沟通、欺骗与协作行为。

**技术手段：**基于多模型智能体生成博弈日志，将其转化为结构化事件流，并通过时间轴与动画引擎实现可加速的阶段化回放与可视化。

**预期结果：**形成一个面向博弈论与专家辩论研究的多智能体行为分析平台，直观展示策略演化、信念变化与模型差异。

\

5️⃣**数字经济周刊智能编辑**

**核心功能与应用场景：**展示近一周内，与数字经济相关的，国内外发布的政策法案、发生的热点新闻。相比于现有的主流报刊、媒体、公众号等，结合领域知识，搜集更加聚焦于数字经济相关主题的内容。

**预期目标：**实现或达到现有博士生同学的搜集、整合、归纳、编辑水平，并测试能否在发布给现有读者群体后不被发现。

\

**6️⃣地球物理反演图辅助解释智能体（看图说话智能体）**

**核心功能与场景：**面向地质反演结果图与科研场景，通过“PDF文献图文清洗”与“反演图像识别”双流处理，利用思维树。对反演结果图进行多模态深度推理与专业解释。

**预期目标：**构建具备文献理解与读图能力的AI助手，实现反演结果的自动化、高精度辅助解释，显著提升科研效率。

\

**7️⃣单细胞注释标记物自动化挖掘智能体（Single-Cell Marker Agent）**

核心功能与应用场景：针对单细胞自动化注释工具质量参差、人工文献调研工作量大的挑战，构建智能体自动化检索PubMed等数据库。用户输入物种、组织、年龄及健康状态后，智能体从海量文献中挖掘Biomarker，并以结构化表格形式整理输出。

预期目标：解决文献异质性与细粒度注释挑战，提供具备证据溯源的高信度标记物列表，显著提升单细胞标记物整理的效率与可靠性。

\

8️⃣**RemoteSense-Analyst：城市遥感领域科研趋势分析研判智能体**

**核心功能：**对城市遥感领域内最新论文进行自动解析，提取最新的研究子方向、方法类型等；基于以上信息，定期生成阶段性研究趋势分析报告；后期引入AI scientist自动模拟投稿和审稿流程。

**应用场景：**分析报告可供科研人员对行业趋势进行快速了解和掌握，启发新思路；经验证后可对新论文思路进行初步review

\

**9️⃣Carbon Universe（碳域宇宙）**

我们打造Web3+AI驱动的合规虚拟碳交易所，直击中小微企业碳资产闲置、个人低碳积分无价值闭环的痛点。依托香港数码港政策，实现中小微碳资产代币化交易、个人低碳行为金融化变现。预期激活千亿级闲置碳资产，实训营后申请数码港专项资金完成香港合规落地。我们以“让每份低碳努力都有价值”为使命，愿景是通过普惠碳金融推动全民参与碳中和，助力1亿人投身双碳行动。
