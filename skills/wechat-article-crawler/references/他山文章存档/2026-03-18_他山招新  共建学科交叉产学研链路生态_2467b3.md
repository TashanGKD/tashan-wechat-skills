**关于他山**

中国科学院大学他山学科交叉创新协会（下文简称“他山协会”）成立于2023年11月，是由中国科学院大学学位办公室、中国科学院物理研究所研究生部、中国科学院大学前沿交叉科学学院指导，中国科学院大学团委备案的学生组织。协会以“打破学科壁垒，拓展认知边界”为核心理念，旨在为在校大学生群体提供跨学科交流与合作的平台。

**以硬科普为核心的跨学科交流**

硬科普是跨学科交流的核心环节。他山协会打造“内部跨学科组会+硬科普文章写作+学科交叉沙龙”系列活动，以“硬科普”为特色，面向不同领域科学家、研究生等具备一定科学素养人群，定期开展内部跨学科学术交流活动，并与中国科学院多个研究所联合举办大型学科交叉沙龙。截至目前，已举办16期内部硬科普报告、8场国科大学科交叉沙龙与2场产学研论坛，共吸引来自40多个院所的60余位专家学者和500余位青年学生参与。

以“落地”为导向的产学研实践

他山协会积极探索产学研融合新模式。自成立以来，在各项活动的开展中，获得了学校及科研院所多部门的支持，并与四十余所高校及科研机构的学生会、学生社团、课题组，以及十余家企业和投资机构建立合作，已主导或参与了八项产业项目，组织和推动了三个垂类科学大模型的建设；首次提出Agent for Science（智能体驱动的科学研究）概念，并形成实现路径；自研MCP-RAG（未来将赋能AI安全建设），打造AI系列课程。

\

作为产学研融合的纽带，他山协会已邀请多位来自产业界和学术界的导师，鼓励并支持有能力的社员成立兴趣小组，围绕产业与科研需求开展有针对性的实践。对于具备转化潜力的项目，他山协会还将提供孵化支持。

**02**

**加入他山**

**在他山，你将收获**

一个拓展视野与思维边界的交叉学科平台

你将走向更广阔的学科版图。通过与不同领域朋友的交流与探讨，拓宽学科视野，增强跨学科思维。思想碰撞擦起灵感火花，迈出打破学科壁垒的第一步。

链接科研资源，促进学术合作与进展

他山汇聚了丰富的科研资源，为成员提供跨学科合作的契机。通过与不同研究方向的研究者的交流合作，你将获得新的研究方法与视角，获取先进的科研生产力工具，推动自身科研工作的深化与突破。

产研结合的实践与探索空间

你可以与兴趣小组成员共同以项目为导向开展学习与研究。依托产研资源的支持，探索技术与应用的结合，推动“智能体+科研”的落地实践，最终实现贯穿科研全流程的AI科学家。

共同建设、共同成长的学术社群

你将有机会深度参与协会的各类活动与建设。从策划到落地，从构想到实践，你不仅是参与者，更是推动者与创造者。在帮助协会发展建设的过程中实现自己的理想。

\

**他山期待这样的你**

\

我们希望你对所选择的项目组有浓厚的兴趣，享受“把事情做得更漂亮”的过程，开放好学、责任心强、自我驱动、乐于分享；我们希望你接受实际工作落地过程中的试错与迭代，愿意透过现象追寻本质问题，追求极致、务实敢为、开放谦逊、坦诚清晰；我们希望你每周有固定、足够的时间投入在小组工作上，在深度参与小组协作中收获个人的迭代和成长。

**他山招新项目组**

\

他山协会现有“办公室”“科普科教项目”“产研实践项目组”三个基本组成单元。其中，“产研实践项目组”以促进跨学科交流为目的开展实践，当前聚焦人工智能主题，设“Agent4S开源项目组”“AIGC项目组”“AI宣讲团”三个项目组。

![](https://mmbiz.qpic.cn/mmbiz_png/YfcILJIWTSicRzI2d7qMrW9yMERS41NKibibfY4iaozOACeUTExdoOAK0648gKTAOxaiaqY3WjqYmAAJjwgJricamiaiaQ/640?wx_fmt=png&from=appmsg)

办公室

部门简介：

办公室是整个社团的“中枢神经”，负责统筹协调各大项目组的运作，是连接社团愿景与日常执行的核心枢纽。

\

除承担行政管理、信息传达、活动备案、财务记录等基础工作外，他山协会办公室正高度模拟现代企业运营，推动社团的数字化转型 ——利用多维表格、数据看板、自动化流程，让每个项目组的协作更高效、更可视、更智能。

\

岗位要求：

1.对组织管理、团队协作、活动策划、数字化工具（如飞书、Notion、Excel、AI工具）感兴趣；

2.具备良好的逻辑分析能力，能将复杂事务拆解、整理、执行，喜欢逻辑清晰、结构有序的工作方式；

3.能清晰表达、有效协调，懂得在不同部门之间建立桥梁；

4.能主动发现问题并尝试解决，有探索精神和持续改进意识。

\

科普科教项目组

部门简介：

科普科教项目组负责他山协会科普与科教相关活动的组织、策划与执行，致力于推动科学传播与科学教育的融合发展，建设具有特色的“他山科普科教”品牌体系。项目组围绕“他山硬科普”“他山科教”等品牌方向，开展一系列面向科研人员与公众的科学传播与教育活动。

\

项目组的核心目标是：\

对外，打造具有影响力的他山科普科教品牌，提升学术传播与科学普及的社会效应；\

对内，面向科研群体，开展科学思维培育，研究技能强化相关活动，推动跨学科交流与科研合作。

\

目前项目组的主要活动包括：中国科学院大学学科交叉沙龙、内部硬科普报告、跨学科课题合作及科普科教文章创作等，持续探索科学传播与学科交叉教育的创新路径。

\

\

岗位要求：

1.组织管理类：负责学科交叉沙龙为主的活动的组织、策划、执行等。要求具备较强的责任心与组织协调、团队协作能力，能够执行工作计划并对问题进行反馈。对学科交叉、科普科教活动有浓厚兴趣，能够在活动周期内投入相当工作时间。在文案撰写、活动组织、直播、摄影及电教设备使用、对外联络等方面有专长者优先。

2.内容编辑类：负责科普科教项目组科普内容的产出、审核、修改。要求对学科知识、学科交叉、科普写作等有浓厚兴趣，具备较好的科学素养与写作、表达能力。

3.平台运营类：负责科普素材面向公众的转化。要求对媒体运营有浓厚的兴趣。熟悉微信公众平台图文编辑，或微信视频号、抖音、B站、知乎等社交平台的运营经历是加分项。

4.视觉设计类：负责科普活动的视觉设计。要求拥有平面设计的基础和兴趣，熟悉图文编辑器使用、熟练使用图片处理软件是加分项。

\

Agent4S开源项目组

部门简介：

Agent4S开源项目组的核心定位是围绕科学垂直领域搭建协作生态，聚焦科研场景下的效率提升与资源整合，通过工具、平台与生态的结合，降低科研工作者在智能体应用中的技术门槛，推动“智能体+科研”的落地实践，最终实现贯穿科学研究全流程的AI科学家。

\

针对Agent for Science领域，他山推出TaShan AgentKit工具套件，构建以 “科学大模型 + 科学 MCP” 为核心的科研生态，在项目落地过程中与阿里云合作密切。

\

项目组三大核心组件：

（1）依托国科大科教平台实现的Agent Builder低代码工作流设计平台；

（2）自研可嵌入IDE的科研扩展TashanCli，实现高效的文献、手稿交互与写作；

（3）自研他山MCP自动化评测智能体，已构建了庞大数据集并可通过多维评价体系实现工具开发闭环。

\

岗位要求：

1.对 Agent for Science 有强烈好奇心，并希望将自己科研领域的工作进行智能化转型；不仅关注技术本身，更愿意探索 “如何真正解决科研实际问题”；

2.只要求基本的 python 代码基础，可使用低代码开发平台做快速验证；可以没有 agent 基础，但要求较高的学习能力，结合他山内部课程快速上手；

3.接受实际工作落地过程中的试错与迭代；

4.保证每周有固定、足够的时间投入在小组工作上。

\

AIGC项目组

部门简介：

AIGC项目组利用AI大模型以及各种智能体工作流平台（Coze、LibLibAI、魔搭社区等）进行AIGC科普科教工作流和AI智能体的搭建，并且利用AI技术实现特定场景（如极宏观、极微观）或特定需求（如大型科学仪器讲解）下的科普科教视频创作。

\

他山AIGC团体是一支以“科学指导+艺术指导”为模式的科普科教内容创作团队，具有较高的科学素养和丰富的科普视频创作经验。同时与众多AI社区（way to AGI、魔搭社区等）有着深度合作关系，并作为部分AIGC比赛的联合发起人，具有丰富的合作资源。加入AIGC项目组，你将获得AIGC技术智能体搭建技术培训与项目的深度实践，以及科普科教视频的创作产出与各种AI科普比赛获奖的机会。

\

岗位要求：

1.对科普科教具有浓厚兴趣，希望可以将自己的研究领域中有趣的科学现象介绍给更多人；

2.对于AIGC技术（主要是视频和图像生成）具有强烈学习意愿，希望掌握相关技术；

3.具有以下技能之一：具有一定的科普科教经验，能够以很好的角度呈现不同的科学内容，将科学知识生动化、通俗化；具有一定的AIGC创作经验，可以利用大模型或者工作流实现科普视频创作；具有一定的视频剪辑、配音方面的经验，可以对AI生成的视频片段进行剪辑。

\

AI宣讲团

部门简介：

AI宣讲团依托他山协会在Agent for Science领域的研究与实践积累，旨在推动AI赋能科研教育的普及与落地。

\

当前主要负责以下工作：

（1）课程体系建设：梳理并输出面向科研场景的 AI通识教育课程体系，内容覆盖大模型基础、智能体开发、科研场景应用等；

（2）平台与活动运营：依托课程体系，策划并组织 国科大AI科教平台 的运营推广，承办 AI实训营、科研创新大赛、讲座工作坊等活动；

（3）学术传播与宣讲：面向科研人员与学生群体，进行AI科研工具、方法论与案例的科普宣讲，推动科研智能化转型。

\

AI宣讲团以科研为导向，结合真实科研场景，致力于形成“AI + Science”教育闭环；同时宣讲团以项目驱动，成员可深度参与AI科研实训营、智能体项目与学术传播活动。宣讲团具有多维的机会，成员可参与AI for Science大赛培训活动组织、课程体系共创等。

\

岗位要求：

1.对 AI与科研结合有强烈好奇心与探索欲；对教学设计、活动策划、内容创作、媒体传播等领域有兴趣；

2.想亲身参与 AI科研实践与教育创新的未来型青年；

3.具备以下技能之一：课程设计能力，能将知识结构化、形成体系化课程内容；活动策划与执行，具备活动组织、项目管理或协调能力；运营与传播，熟悉新媒体内容传播、社群运营或品牌设计。
