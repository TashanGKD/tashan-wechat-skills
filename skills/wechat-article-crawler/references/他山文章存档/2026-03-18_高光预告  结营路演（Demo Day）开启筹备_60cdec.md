# 高光预告 | 结营路演（Demo Day）开启筹备

![](https://mmbiz.qpic.cn/mmbiz_jpg/YfcILJIWTS8tPTFTJz9RECoT8NJKWQoAPlleDJnhzt5qc0RsH0cBCzmRYDYBLSFgDHBH2171icTdSlClWO82S0A/640?wx_fmt=jpeg)

🎤 高光预告

随着实训营进入中后期，实训营将逐步从“学习阶段”过渡到「成果沉淀 + 展示准备」阶段。我们最激动人心的环节——结营路演 即将拉开帷幕！12月底结营阶段，我们将组织**结营路演**（Demo Day）， 集中展示大家在实训营中的学习成果。这不仅是展示你四周学习成果的舞台，更是将你的“AI科研伙伴”推向更广阔天地的机会。

**路演展示什么？**

\

\

\

路演将围绕以下两类成果展开（个人或团队均可）：

✅**学科专属“AI科研伙伴”**（ AI Scientist / AI 科研应用）围绕你自己的研究方向，做出可演示的成果，例如：

1️⃣文献/数据的自动整理与抽取

2️⃣可复用的分析流程与科研工作流

3️⃣一个能完成具体科研任务的工具或助手

（如整理、分析、生成报告等）

✅经管**领域专项赛道项目** 围绕反垄断、数字经济、商业与管理等方向的专题应用

（文件请参考群通知，视频回放如下）

\

**谁可以参加路演？**

\

\

\

👉 路演面向实训营**入营学员**

👉 采取**自愿报名**方式

👉 最终路演名单将**综合参考**：

1️⃣路演报名意向

2️⃣四次课程作业的完成情况

3️⃣作品成熟度与展示节奏

🎯 **Tips：**路演更看重“你如何把问题做成方法”，而不是一次性做出完美成品。

**路演支持方式**

\

\

\

📖**作业：** AI 自动评价 +问题在线收集

❓**答疑**：由主讲老师在群内进行集中答疑/统一回应

💪Week 4（暂定12月27日周六）将组织：

1️⃣集中技术答疑

2️⃣路演结构与展示节奏讲解（模板/示例）

3️⃣Demo 打磨方向建议（公开辅导形式）

**激励与长期发展机会（预告）**

\

\

\

结营路演将设置多层次激励与长期支持（覆盖多支团队），包括官方认证、平台展示与作品收录等。**具体评选规则与激励细则将在临近结营时统一公布，请以群内正式通知为准。**

\

同时，我们将持续强调“成果的长期价值”：

🏆优秀作品将获得平台**展示与收录**机会（如国科大AI科教平台、阿里云科教专区等平台）

🏆优秀项目有机会进入后续**共创与发展**通道（文章延伸指导/项目合作/赛事推荐等，具体以主办方后续安排为准）
