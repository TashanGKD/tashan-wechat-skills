## §3 颜色规范

```
主题绿：rgb(111,167,170)
用途：H2 标题、核心 H3、底部署名标签、协会简介、二维码提示文字
限制：全文绿色不超过 15 处

strong 颜色：rgb(62,62,62)（深灰，非黑色）
```

## §5 发布核查清单（发布前必查）

```
HTML 格式检查：
  □ 无 ASCII 残留字符（引号、破折号等）
  □ 图片数量与配图索引一致
  □ 绿色用量 ≤ 15 处
  □ strong 颜色为 rgb(62,62,62)，非纯黑
  □ H2 标题为绿色
  □ 底部模板完整（logo+二维码+协会简介）
  □ 署名正确

内容检查（由 article-proofreading 负责）：
  □ 无 AI 腔（「值得注意的是」「不可否认」等）
  □ 标题不含绝对表达
  □ 结语有实质内容
```

## §6 已知注意事项

| # | 问题 | 解决方式 |
|---|------|---------|
| W-01 | HTML 转换时引号变为 ASCII | 转换后必须检查并替换 |
| W-02 | 底部图片文件名与用途相反（高频坑） | 永远记用途不记文件名（见§2：`qrcode_candidate.png` 是 logo，`last.jpg` 是二维码） |
| W-03 | 绿色超出 15 处导致「绿色泛滥」视觉 | 写完后 Ctrl+F 统计绿色数量 |
